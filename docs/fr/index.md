# Assistance Beanfolio

Bienvenue sur la page d'assistance de Beanfolio. Nous sommes là pour vous aider à tirer le meilleur parti de votre compagnon café de spécialité.

## Contactez-nous
Si vous avez des questions, rencontrez un bug ou avez des suggestions de nouvelles fonctionnalités, n'hésitez pas à nous contacter :

**E-mail :** [barista@beanfolio.app](mailto:barista@beanfolio.app)

## Foire aux questions (FAQ)

### Comment fonctionne le scanner ?
Beanfolio utilise l'IA pour extraire les informations du recto et du verso des sachets de café. Pour obtenir les meilleurs résultats, assurez-vous que le sachet est bien éclairé et que le texte est clairement visible dans le cadre.

### Puis-je modifier les informations après le scan ?
Oui ! Après le scan, vous accéderez à un écran de révision où vous pourrez appuyer sur n'importe quel champ pour modifier les détails extraits avant de les enregistrer dans votre collection.

### Où mes données sont-elles stockées ?
Votre collection de café est stockée en toute sécurité dans le cloud via Firebase, ce qui signifie que vous pouvez y accéder depuis n'importe quel appareil sur lequel vous êtes connecté avec le même compte.
